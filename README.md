# FlightRadar V2

Mobilanpassad PWA för iPhone som visar flyg runt din aktuella position.

## Starta via GitHub Pages

1. Öppna repositoryts **Settings**.
2. Välj **Pages**.
3. Under **Build and deployment**, välj **Deploy from a branch**.
4. Välj branch **main** och mappen **/(root)**.
5. Spara.
6. Öppna: `https://fartingdad.github.io/FlightRadar/`

På iPhone:
1. Öppna länken i Safari.
2. Tillåt platsåtkomst.
3. Tryck **Dela**.
4. Välj **Lägg till på hemskärmen**.

## V2-funktioner

- Futuristisk radarvy anpassad för iPhone
- GPS-position och valbar radie 5–250 NM
- Airplanes.live med ADSB.lol som reserv
- Automatisk uppdatering var 10:e sekund
- Filter för höjd, fart och datatyp
- Callsign- eller registreringsetiketter
- Lokala flygspår från senaste uppdateringarna
- Följ valt flyg
- Detaljpanel med höjd, fart, kurs, vertikalhastighet, typ, registrering och squawk
- Installerbar PWA
- Offline-cache för appens gränssnitt

## Observera

Internet krävs för karta och flygdata. Täckningen beror på tillgängliga ADS-B/MLAT-mottagare. Appen är för hobby och information, inte navigation eller flygsäkerhetskritisk användning.
